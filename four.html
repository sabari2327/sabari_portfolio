Interface DHT11 Sensor with Arduino

Objective

To interface DHT11 temperature and humidity sensor with Arduino and display the readings on Serial Monitor.

Components

  • 1 x Arduino UNO
  • 1 x DHT11 Sensor
  • 1 x Breadboard
  • Jumper Wires

Connections

  • VCC - 5V
  • GND - GND
  • DATA - A0

Arduino Code

#include <DHT.h>

#define DHTPIN A0
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();

  Serial.print("Humidity: ");
  Serial.print(h);
  Serial.print("% ");

  Serial.print("Temperature: ");
  Serial.print(t);
  Serial.println(" *C");

  delay(2000);
}

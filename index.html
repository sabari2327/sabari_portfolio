Interface Push Button with Arduino to Control LED

Press the button to turn ON the LED

Objective

To design and implement an Arduino-based system that interfaces with a push button to control an LED. The LED turns ON when the button is pressed.

Components

  • 1 x Arduino UNO
  • 1 x Breadboard
  • 1 x Red LED
  • 1 x Push Button
  • 1 x 1K Resistor
  • Jumper Wires

Connections

  • Pin 9 - LED Anode (+)
  • Pin 13 - Push Button
  • GND - Common Ground
  • Resistor - 1K between LED and GND

Arduino Code

#include <DHT.h>

#define DHTPIN A0
#define DHTTYPE DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();

  Serial.print("Humidity: ");
  Serial.print(h);
  Serial.print("% ");

  Serial.print("Temperature: ");
  Serial.print(t);
  Serial.println(" *C");

  delay(2000);
}
